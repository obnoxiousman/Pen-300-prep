**Disable Windows Defender**
```cmd
sc.exe config WinDefend start= auto
sc.exe start WinDefend
sc.exe stop WinDefend
```

**Disable Threat Protection**
```cmd
Set-MpPreference -DisableRealtimeMonitoring $true
```
