**MSSQL Enumerate DB Users**
```SQL
SELECT name AS [Login Name], type_desc AS [Account Type] FROM sys.server_principals WHERE TYPE IN ('U', 'S', 'G') and name not like '%##%' ORDER BY name, type_desc
```
